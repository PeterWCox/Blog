Welcome to my blog! This is a minimal blog built with [Astro](https://astro.build) and hosted on [GitHub Pages](https://pages.github.com).

## How to Write Posts

To create a new blog post, simply create a new Markdown file in the `src/content/blog/` directory with the following frontmatter:

```markdown
---
title: Your Post Title
description: A brief description of your post
pubDate: 2024-12-30
---

Your content here...
```

## Features

- **Markdown Support**: Write your posts in Markdown
- **Automatic Deployment**: Push to GitHub and your site updates automatically
- **Clean Design**: Minimal, responsive theme
- **Fast**: Built with Astro for optimal performance

## Getting Started

1. Create a new `.md` file in `src/content/blog/`
2. Add the frontmatter with title, description, and pubDate
3. Write your content in Markdown
4. Commit and push to GitHub
5. Your post will be live within minutes!

Happy writing!
